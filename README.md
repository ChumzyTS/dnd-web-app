# DnD Web App
A D&D 5.5e character sheet web app.

## Frontend
### Framework
- Vite
- React

### Running the Frontend
**1. Install Dependencies**
(Can skip if already done)
- `cd .\frontend\`
- `npm install`

**2. Run it**
- `npm run dev`
